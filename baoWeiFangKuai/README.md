## 源码目录介绍
```
./js
├── base                                   // 定义游戏开发基础类
│   ├── animatoin.js                       // 帧动画的简易实现
│   ├── pool.js                            // 对象池的简易实现
│   └── sprite.js                          // 游戏基本元素精灵类
├── bullet
│   └── bullet.js                          // 子弹类
├── enemy
│   └── enemy.js                           // 敌机类
├── player
│   └── player.js                          // 玩家类
├── UI
│   └── UI.js                              // 用于展示分数和血量
├── page
│   ├── gameOver.js                        // 游戏结束页面渲染对象
│   └── gameStart.js                       // 游戏开始页面渲染对象
├── background.js                          // 背景类
└── main.js                                // 游戏入口主函数

```
